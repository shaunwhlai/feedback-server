***In order for this project to work properly, you have to run "Start MongoDB" bat file to execute MongoDB so that the data inputted by the user can be stored in MongoDB. Otherwise the program will just crash.***

Execute these commands in terminal.
>>cd C:\Program Files\MongoDB\Server\4.0\bin
>>mongod.exe --dbpath D:\mongo-data

1) navigate to project folder and ">>npm start" in terminal

2) Please use the url "http://localhost:3000/feedback" to view my feedback form. 

This feedback form request user to input their Full Name, Email, Mobile, City and Comments and all this data will be stored in MongoDB and it will show the user that the data has been stored in the server.
